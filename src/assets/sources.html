Hammer Units To Kilometers Conversion Method

  • Initial research lead to no single way to convert hammer units to normal measurements, sizes can apparently vary from game to game
  • Opted to use the following pastebin link mentioned in the description of Uncle Dane's video on map sizes
    • https://pastebin.com/rwXevdPs
  • Pastebin file only shows the before and after conversion numbers, does not show conversion method.
  • Deduced that the conversion method is the following (May be incorrect, will need to do additional research to verify this):
    • Hu^2 / 27.5926 = km^2
